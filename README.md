# Discord-uWuFication
Discord-uWuFication is a small discord webhook written in PHP to parse an RSS feed trough and 'UwUfy' the contents, after this it gets posted in Discord.
## Requirements
- PHP 7.3+ 
- PHP-xml
- PHP-curl
